Volley is a competitive 2v2 volleyball game featuring physics-based ball mechanics, team-based scoring, and bot opponents.

## Gameplay

- **Teams**: 2v2 (Team 1 vs Team 2)
- **Objective**: Hit the ball over the net to score points
- **Win Condition**: First to 7 points
- **Mechanics**: Jump serves, spike attacks, 3-hit limit per team
- **Bots**: AI players fill empty slots

### Point Phase

Location: `core/src/games/volley/Volley.ts:154`

```typescript
if (state.phase === "point") {
  // Wait for ball to land
  if (ballPos.data.z > 0) return

  state.lastWinTick = world.tick

  // Update score
  if (state.lastWin === 1) state.scoreLeft++
  if (state.lastWin === 2) state.scoreRight++
  state.teamServing = state.lastWin === 2 ? 2 : 1

  // Reset to 0-0 if either team reaches 7
  if (state.scoreLeft >= 7 || state.scoreRight >= 7) {
    state.scoreLeft = 0
    state.scoreRight = 0
  }

  // Transition to serve
  state.phase = "serve"
  state.lastHit = ""
  state.lastHitTeam = 0
  state.hit = 0

  // Reset ball
  ballPos.setVelocity({ x: 0, y: 0, z: 0 })
  ballPos.setRotation(0)
  ballPos.setGravity(0)
  ballPos.setPosition({
    x: state.teamServing === 1 ? 10 : 400,
    y: 1, z: 50
  })
}
```

## Actions

### Spike

Location: `core/src/games/volley/Spike.ts`

Hit the ball toward target:

```typescript
export const Spike = () => Action("spike", ({ entity, world, params }) => {
  const { position } = entity.components
  const state = world.state<VolleyState>()
  
  // Track consecutive hits
  if (state.lastHitTeam === team) {
    state.hit++
    if (state.hit > 3) return // Max 3 hits per team
  } else {
    state.hit = 1
  }
  
  // Track jump hits (can't hit twice while airborne)
  if (!position.data.standing) {
    if (state.jumpHits.includes(entity.id)) return
    state.jumpHits.push(entity.id)
  }
  
  // Calculate velocity toward target
  const dx = params.target.x - ballPos.data.x
  const dy = params.target.y - ballPos.data.y
  const distance = hypot(dx, dy)
  
  const speed = 400 // Base spike speed
  ballPos.setVelocity({
    x: (dx / distance) * speed,
    y: (dy / distance) * speed,
    z: calculateZ(distance) // Lob height based on distance
  })
  
  state.lastHit = entity.id
  state.lastHitTeam = team
  state.lastHitTick = world.tick
})
```

## Key Features

### Rollback Netcode

Volley uses rollback for competitive play:
- Low-latency input
- Client-side prediction
- State reconciliation

### Dynamic Bot Management

Bots automatically fill teams:
- Spawn when players < 4
- Despawn when players join
- Maintain 2v2 balance

### Physics-Based Ball

- Gravity: 0.3
- Collision with net
- Rotation based on velocity
- Bounce mechanics
