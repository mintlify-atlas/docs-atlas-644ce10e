Strike is a competitive first-person shooter featuring team-based gameplay, voxel maps, and rollback netcode for low-latency combat.

## Gameplay

- **Teams**: 5v5 (Attackers vs Defenders)
- **Perspective**: First-person with Three.js rendering
- **Maps**: Retake-style voxel maps
- **Weapons**: Inventory system with different guns
- **Rounds**: Warmup, round start, play, end phases
- **KDA**: Kill/Death/Assist tracking

## Systems

### StrikeSystem

Location: `core/src/games/strike/Strike.ts:82`

```typescript
const StrikeSystem = SystemBuilder({
  id: "StrikeSystem",
  init: (world) => {
    // Load voxel map
    world.blocks.loadMap(RetakeMap)
    world.blocks.coloring = RetakeMapColoring

    const mobileUI = MobileUI(world)

    // Set initial camera aim
    if (world.client) world.client.controls.localAim.x = 0

    return {
      id: "StrikeSystem",
      query: [],
      priority: 3,
      onTick: () => {
        const state = world.state<StrikeState>()
        const settings = world.settings<StrikeSettings>()

        // Menu control (desktop only)
        if (world.client && !world.client.mobile) {
          world.client.menu = document.pointerLockElement === null
        }

        mobileUI?.update()

        const players = world.players()

        // Warmup phase logic
        if (world.mode === "server" && state.phase === "warmup") {
          const pcs = players.filter(p => !p.id.includes("dummy"))
          const ready = players.filter(p => p.components.pc.data.ready)

          // Start when all players ready
          if (ready.length && ready.length === pcs.length) {
            state.phaseChange = world.tick + 120
          }
        }

        // Phase transitions
        if (state.phaseChange && world.tick >= state.phaseChange) {
          if (state.phase === "warmup") {
            state.phase = "round-start"

            // Spawn players at team positions
            let i = 0, j = 0
            for (const player of players) {
              const character = player.components.controlling?.getCharacter(world)
              if (!character) continue

              const { position, health } = character.components
              const teamNumber = player.components.team.data.team
              const randomPoint = spawnPoints[teamNumber][teamNumber === 1 ? i++ : j++]

              position.setPosition(randomPoint)
              position.data.aim.x = randomPoint.r

              if (health) health.setKDA({ k: 0, d: 0, a: 0 })

              if (world.client?.playerId() === player.id) {
                world.client.controls.localAim.x = randomPoint.r
              }
            }
          }
          state.phaseChange = null
        }

        // Jump state cleanup
        for (const player of players) {
          const character = player.components.controlling?.getCharacter(world)
          if (!character) continue

          const { position, health } = character.components
          if (!health) continue
          const { standing, velocity } = position.data

          if (standing && velocity.z <= 0) {
            state.jumped = state.jumped.filter(id => id !== character.id)
          }

          // KDA tracking on death
          if (health?.dead() && health.data.died === world.tick - 10) {
            const { k, d, a } = health.getKDA()
            health.setKDA({ k, d: d + 1, a })

            // Award kill to attacker
            const fromPlayer = world.entity(health.data.diedFrom || "")
            const fromCharacter = fromPlayer?.components.controlling?.getCharacter(world)
            if (fromCharacter?.components.health) {
              const kda = fromCharacter.components.health.getKDA()
              fromCharacter.components.health.setKDA({ 
                k: kda.k + 1, d: kda.d, a: kda.a 
              })
            }
          }

          // Reset rotation
          position.data.rotating = 0
          if (rotation < 0) position.data.rotating = min(0.08, -rotation)
          if (rotation > 0) position.data.rotating = -1 * min(0.08, rotation)

          // Respawn if fell off map
          if (z < -4) {
            position.setPosition({ x: 9.9, y: 15, z: 2 })
          }
        }
      }
    }
  }
})
```

## Game Phases

### Warmup Phase

Location: `core/src/games/strike/Strike.ts:109`

- Players move freely
- No objectives active
- Waiting for all players to ready up
- Transitions when all ready

### Round Start Phase

Location: `core/src/games/strike/Strike.ts:118`

- Players spawn at team positions
- KDA reset
- Brief countdown
- Transitions to play phase

### Round Play Phase

- Normal gameplay
- Combat active
- Objective pursuit
- Ends when objective complete or time expires

### Round End Phase

- Shows winners
- Score update
- Brief pause before next round

## Key Features

### Rollback Netcode

Strike uses rollback for competitive FPS:
- Low input latency
- Hit registration accuracy
- State reconciliation

### Voxel Maps

- Fully destructible (optional)
- Custom block coloring
- Efficient chunk rendering
- Multiple map support

### First-Person Camera

- Pointer lock control
- Mouse sensitivity settings
- First/third person toggle
- FOV customization

### Inventory System

- Multiple weapons
- Reload mechanics
- Weapon switching
- Ammo management

### Team System

- 5v5 gameplay
- Spawn points per team
- Team switching (T key)
- Scoreboard with teams
