Mars is a rocket launching simulation game where players manage resources, build rockets, and launch them toward Mars.

## Gameplay

- **Objective**: Successfully launch rockets to Mars
- **Resources**: Money management system
- **Time**: Date progression (days/months/years)
- **Revenue**: FarLink income and contract payments
- **Expenses**: Rocket component costs
- **Launch States**: Unready, ready, firing, floating

## Systems

### MarsSystem

Location: `core/src/games/mars/Mars.ts:30`

```typescript
const MarsSystem = SystemBuilder({
  id: "MarsSystem",
  init: (world) => {
    const mobileUI = MobileUI(world)
    let lastDayTick = world.tick

    return {
      id: "MarsSystem",
      query: [],
      priority: 2,
      onTick: () => {
        mobileUI?.update()
        const state = world.state<MarsState>()

        // Advance time
        if (world.tick - lastDayTick >= ticksPerDay 
            && ["ready", "unready"].includes(state.readiness)) {
          lastDayTick = world.tick
          state.date = nextDay(state.date)
        }
      }
    }
  }
})
```

Manages:
- Day/night cycle
- Date progression
- Mobile UI updates
- Time-based events

#### LaunchButton

Location: `core/src/games/mars/ui/LaunchButton.ts`

Initiate rocket launch:

```typescript
export const LaunchButton = (): Entity => Entity({
  id: "launch-button",
  components: {
    html: HButton({
      text: "LAUNCH",
      style: {
        position: "fixed",
        bottom: "40px",
        left: "50%",
        transform: "translateX(-50%)",
        width: "200px",
        height: "60px",
        fontSize: "24px",
        backgroundColor: "#ff4444",
        color: "white",
        border: "3px solid white",
        borderRadius: "10px"
      },
      onRelease: ({ world }) => {
        const state = world.state<MarsState>()
        if (state.readiness === "ready") {
          world.actions.push(world.tick + 1, "rocket", { 
            actionId: "launch" 
          })
        }
      }
    }),
    npc: NPC({
      behavior: (entity, world) => {
        const state = world.state<MarsState>()
        const button = entity.components.html?.div as HTMLButtonElement
        
        if (button) {
          button.disabled = state.readiness !== "ready"
          button.style.opacity = state.readiness === "ready" ? "1" : "0.5"
        }
      }
    })
  }
})
```

## Launch States

Location: State management

### Unready

- Rocket being built
- Cannot launch
- Resources being spent
- Time passes normally

### Ready

- Rocket fueled and prepared
- Launch button enabled
- Waiting for player input
- Time passes normally

### Firing

- Rocket engines active
- Accelerating upward
- Time paused
- Visual effects active

### Floating

- Reached orbit
- Zero gravity
- Mission complete
- Waiting for reset
