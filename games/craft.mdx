Craft is a survival game featuring procedurally generated voxel terrain, tree spawning, building mechanics, and a unique "eagle vs duck" game mode where one player hunts others.

## Gameplay

- **Mode**: Eagle vs Duck survival
- **Terrain**: Procedurally generated grass, trees, and hills
- **Building**: Place and break voxel blocks
- **Objective**: Ducks survive while eagle hunts them
- **Flying**: Eagles can fly; ducks cannot
- **Rounds**: Multi-round with rotating eagle player
- **Points**: Eagle earns 3 points per duck caught

## Systems

### CraftSystem

Location: `core/src/games/craft/Craft.ts:79`

```typescript
const CraftSystem = SystemBuilder({
  id: "CraftSystem",
  init: (world) => {
    // Generate initial terrain
    spawnTerrain(world, 24)

    const mobileUI = MobileUI(world)
    const preview = BlockPreview(world)
    if (preview) world.three?.scene.add(preview.mesh)

    return {
      id: "CraftSystem",
      query: [],
      priority: 3,
      onTick: () => {
        const state = world.state<CraftState>()
        const settings = world.settings<CraftSettings>()
        const players = world.players()
        const characters = world.characters()

        // Pointer lock control
        if (world.client && !world.client.mobile) {
          world.client.menu = document.pointerLockElement === null
        }

        mobileUI?.update()

        // Block preview for placement
        const pc = world.client?.character()
        if (pc && preview) {
          preview.update(world.three!.camera.pos(), world.three!.camera.dir(world))
        }

        // Start game when all players ready
        if (world.mode === "server" && state.phase === "warmup" && players.length > 1) {
          const notReady = players.filter(p => !p.components.pc.data.ready)
          if (notReady.length === 0) {
            state.phase = "starting"
            state.willStart = world.tick + 40 * 3
            state.nextSeed = randomInt(1000000)
          }
        }

        // Check win condition (all ducks caught)
        if (world.mode === "server" && state.phase === "play") {
          const ducks = characters.filter(c => !c.components.position.data.flying)

          if (ducks.length === 0) {
            if (characters.length === state.startedEagle.length) {
              // Everyone has been eagle - declare winner
              const winner = players[0].components.pc.data.name
              world.announce(`${winner} wins! GG!`)

              state.phase = "warmup"
              state.startedEagle = []
              state.round = 0

              players.forEach(p => {
                p.components.pc.data.ready = false
                p.components.pc.data.points = 0
              })
            } else {
              // Start next round with new eagle
              state.phase = "starting"
              state.willStart = world.tick + 40 * 3
              state.nextSeed = randomInt(1000000)
            }
          }
        }

        // Start round
        if (state.phase === "starting" && world.tick === state.willStart!) {
          world.announce(`round ${state.round + 1}!`)

          state.phase = "play"
          state.round += 1

          // New random seed
          world.random = Random(state.nextSeed)

          // Reset world
          world.blocks.clear()
          world.trees = []
          spawnTerrain(world, 24)

          // Reset player positions
          for (const character of characters) {
            const { position } = character.components
            position.setPosition({ 
              x: 14 + world.random.int(12, 6), 
              y: 14 + world.random.int(12, 6), 
              z: 6 
            })
            position.setVelocity({ x: 0, y: 0, z: 0 })
            position.data.flying = false
          }

          // Choose eagle
          const candidates = characters.filter(c => !state.startedEagle.includes(c.id))
          const eagle = world.random.choice(candidates)
          if (eagle) {
            eagle.components.position.data.flying = true
            eagle.components.position.setPosition({ x: 15, y: 31, z: 6 })
            state.startedEagle.push(eagle.id)
          }
        }

        // Handle player physics
        for (const player of players) {
          const character = player.components.controlling?.getCharacter(world)
          if (!character) continue

          const { position } = character.components
          const { z, rotation, standing } = position.data

          // Reset double jump on landing
          if (standing) {
            state.doubleJumped = state.doubleJumped.filter(id => id !== character.id)
          }

          // Reset rotation drift
          position.data.rotating = 0
          if (rotation < 0) position.data.rotating = min(0.08, -rotation)
          if (rotation > 0) position.data.rotating = -1 * min(0.08, rotation)

          // Respawn if fell off world
          if (z < -4) {
            position.setPosition({ x: 20, y: 20, z: 8 })
          }

          // Eagle catches ducks
          if (world.mode === "server" && position.data.flying) {
            const ducks = characters.filter(c => !c.components.position.data.flying)

            for (const duck of ducks) {
              const duckPos = duck.components.position
              const distance = XYZdistance(position.data, duckPos.data)

              if (distance < 0.24) {
                if (state.phase === "play") {
                  duckPos.data.flying = true
                  player.components.pc.data.points += 3
                  world.announce(`${player.components.pc.data.name} caught a duck!`)
                } else {
                  duckPos.setPosition({ x: 20, y: 20, z: 6 })
                }
              }
            }
          }
        }

        // Spawn apples (collectibles)
        if (!applesSpawned) {
          for (let i = 0; i < 25; i++) {
            world.addEntity(Apple({ id: `d3apple-${1 + i}` }))
          }
          applesSpawned = true
        }
      }
    }
  }
})
```

## Game Phases

### Warmup Phase

- Players join and explore
- Waiting for all players to ready
- Free flight available
- No scoring

### Starting Phase

Location: `core/src/games/craft/Craft.ts:154`

- 3 second countdown (120 ticks)
- Regenerate terrain with new seed
- Choose random eagle from candidates
- Reset all player positions

### Play Phase

Location: `core/src/games/craft/Craft.ts:131`

- Eagle hunts ducks
- Ducks try to survive
- Building and breaking blocks allowed
- Round ends when all ducks caught

## Mechanics

### Eagle vs Duck

**Eagle:**
- Can fly freely
- Spawns at high altitude
- Catches ducks by proximity (< 0.24 distance)
- Earns 3 points per catch

**Ducks:**
- Cannot fly
- Can double jump
- Must evade eagle
- Become eagle when caught
