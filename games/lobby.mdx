The Lobby is Piggo's main menu and game selection hub. It's where players gather before starting a multiplayer session and choose which game to play.

## Gameplay

The Lobby provides:
- Visual game selection with preview images
- Player profile display with animated frames
- Lobbies menu for joining/creating rooms
- Online player count
- Music controls
- Mobile-optimized carousel interface

## Game Starting

Location: `core/src/games/lobby/Lobby.ts:32`

Only the lobby leader can start games:

```typescript
const startGame = (game: GameBuilder, world: World, state: LobbyState) => {
  if (!world.client?.isLeader()) return
  if (state.starting) return

  world.client?.sound.play({ name: "bubble" })
  world.actions.push(world.tick + 20, "world", { 
    actionId: "game", 
    params: { game: game.id } 
  })
  state.starting = true
}
```

The action is delayed by 20 ticks to allow sound to play before transition.
