Math utilities provide core mathematical operations for game development including vector math, coordinate transformations, physics calculations, and collision detection.

## Types

### Position Types

<ParamField path="XY" type="{ x: number, y: number }">
  2D coordinate position
</ParamField>

<ParamField path="XZ" type="{ x: number, z: number }">

</ParamField>

<ParamField path="XYZ" type="{ x: number, y: number, z: number }">
  3D coordinate position
</ParamField>

<ParamField path="XYZR" type="{ x: number, y: number, z: number, r: number }">

</ParamField>

## Basic Math Functions

### minmax

```typescript
minmax(n: number, minValue: number, maxValue: number): number
```

Clamps a number between minimum and maximum values.

<ParamField path="n" type="number" required>
  The value to clamp
</ParamField>

<ParamField path="minValue" type="number" required>
  Minimum allowed value
</ParamField>

<ParamField path="maxValue" type="number" required>
  Maximum allowed value
</ParamField>

```typescript
const health = minmax(damage, 0, 100) // Clamp health between 0-100
```

### round

```typescript
round(n: number, places?: number): number
```

Rounds a number to specified decimal places.

<ParamField path="n" type="number" required>
  Number to round
</ParamField>

<ParamField path="places" type="number" default="0">
  Number of decimal places
</ParamField>

```typescript
const rounded = round(3.14159, 2) // 3.14
```

### reduce

```typescript
reduce(n: number, by: number): number
```

Reduces a number toward zero by a specified amount. Useful for friction or decay.

<ParamField path="n" type="number" required>
  Current value
</ParamField>

<ParamField path="by" type="number" required>
  Amount to reduce by
</ParamField>

```typescript
// Apply friction to velocity
velocity.x = reduce(velocity.x, 0.5)
```

## Vector Operations

### Distance Functions

<Accordion title="Distance Calculations">

#### XYdistance

```typescript
XYdistance(a: XY, b: XY): number
```

Calculates Euclidean distance between two 2D points.

```typescript
const dist = XYdistance({ x: 0, y: 0 }, { x: 3, y: 4 }) // 5
```

#### XYZdistance

```typescript
XYZdistance(a: XYZ, b: XYZ): number
```

Calculates Euclidean distance between two 3D points.

```typescript
const dist = XYZdistance(playerPos, targetPos)
```

#### XYdiff / XYZdiff

```typescript
XYdiff(a: XY, b: XY, threshold?: number): boolean
XYZdiff(a: XYZ, b: XYZ, threshold?: number): boolean
```

Checks if distance between points exceeds a threshold.

```typescript
// Used in Spike action (volley/Spike.ts:14)
const far = XYZdiff(from, ballPos.data, range)
if (far) return // Too far to spike
```

</Accordion>

### Normalization

<Accordion title="Vector Normalization">

#### XYZnormal

```typescript

```

Normalizes a vector using Manhattan distance (sum of absolute values).

#### normalize


#### vectorExtend

```typescript

```

Extends a vector by a specified amount while maintaining direction.

</Accordion>

## Physics Calculations

### Projectile Motion

<Accordion title="Ballistic Functions">

#### timeToLand

```typescript
timeToLand(g: number, z: number, v: number): number
```

Calculates time until a projectile lands given gravity, height, and vertical velocity.

<ParamField path="g" type="number" required>
  Gravity constant
</ParamField>

<ParamField path="z" type="number" required>
  Current height
</ParamField>

<ParamField path="v" type="number" required>
  Vertical velocity
</ParamField>

#### velocityToPoint

```typescript
velocityToPoint(from: XYZ, to: XY, g: number, v: number): XY
```

Calculates horizontal velocity needed to reach a point with ballistic motion.

```typescript
// Used in Spike action (volley/Spike.ts:88)
v = velocityToPoint(ballPos.data, target, g, vz)
ballPos.setVelocity({ z: vz, x: v.x / 25 * 1000, y: v.y / 25 * 1000 })
```

#### velocityToDirection

```typescript
velocityToDirection(from: XYZ, to: XY, distance: number, g: number, v: number): XY
```

Calculates velocity to reach a specific distance toward a target point.

```typescript
// Used in Spike action (volley/Spike.ts:99)
v = velocityToDirection(ballPos.data, target, 80, g, vz)
```

</Accordion>

### Time and Day Cycle

<Accordion title="Time Functions">

#### hourness

```typescript
hourness(tick: number, delta: number): number
```

Calculates current hour in a 24-hour cycle based on game tick.

#### dayness

```typescript
dayness(tick: number, delta: number): number
```

Returns daylight intensity (0-1) based on time of day. Returns 1 during daytime (hours 0-9), fades during dusk (9-12) and dawn (21-24).

</Accordion>

## Usage Examples

### Movement with Physics

```typescript
import { XYdistance, velocityToPoint, minmax } from "@piggo-gg/core"

// Calculate velocity to throw object at target
const from = entity.components.position.data
const velocity = velocityToPoint(from, target, gravity, verticalSpeed)

entity.components.position.setVelocity({
  x: velocity.x,
  y: velocity.y
})
```

### Distance-Based Interactions

```typescript
import { XYZdistance, closestEntity } from "@piggo-gg/core"

// Find nearest enemy within range
const nearestEnemy = closestEntity(
  player.components.position.data,
  enemies,
  attackRange
)

if (nearestEnemy) {
  // Attack nearest enemy
}
```

### Collision Detection

```typescript
import { raySphereIntersect, XYZ } from "@piggo-gg/core"

const rayOrigin: XYZ = { x: 0, y: 0, z: 1 }
const rayDir: XYZ = { x: 1, y: 0, z: 0 }
const sphereCenter: XYZ = { x: 10, y: 0, z: 1 }
const sphereRadius = 2

if (raySphereIntersect(rayOrigin, rayDir, sphereCenter, sphereRadius)) {
  // Ray hit the sphere
}
```

### Random Generation

```typescript
import { randomChoice, randomInt, randomLR } from "@piggo-gg/core"

// Random spawn position with offset
const spawnX = baseX + randomLR(20)
const spawnY = baseY + randomLR(20)

// Random enemy type
const enemyType = randomChoice(["goblin", "orc", "troll"])

// Random damage variation
const damage = baseDamage + randomInt(5)
```
